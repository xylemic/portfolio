This interactive game challenges you to guess a random number between 1 and 50. You get 5 attempts and can request hints for help. The game provides feedback on your guesses (too high or too low) and congratulates you when you win. This project demonstrates skills in: Creating interactive web experiences, Using JavaScript for game logic, Generating random numbers, Displaying messages based on user input.
